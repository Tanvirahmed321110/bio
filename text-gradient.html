How This Effect Works

  • The text uses -webkit-background-clip: text and -webkit-text-fill-color: transparent to show the image through the text.
  • A gradient overlay with animation is applied using a pseudo-element (::after) with mix-blend-mode: overlay.
  • The gradient animation moves the background position to create the flowing color effect.
  • Another pseudo-element (::before) creates a pulsing shadow effect for added depth.
  • The entire text container has a floating animation for additional visual interest.
